*Rápido e devagar* de Kahneman (2011) sintetizou décadas do trabalho dele com Amos Tversky. O Sistema 1 está sempre ligado — lê os rostos, completa as frases, reconhece padrões. O Sistema 2 é relutante e custoso — engaja-se quando o Sistema 1 sinaliza um mismatch ou quando o forçamos.

A aprendizagem é uma atividade do Sistema 2. Mas o Sistema 1 sempre procura atalhos, e um LLM fornece o atalho perfeito: uma resposta tão fluente que o Sistema 2 nunca se engaja. O learner se sente ensinado; nada foi ensinado.

A pedagogia do Fluera é construída para forçar o engajamento do Sistema 2. A escrita à mão é lenta de propósito. A avaliação de confiança interrompe a leitura de superfície. Os prompts socráticos se recusam a ser System-1-ready. O atrito é o habitat do Sistema 2.
