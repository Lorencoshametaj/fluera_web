*Pensar rápido, pensar despacio* de Kahneman (2011) sintetizó décadas de su trabajo con Amos Tversky. El Sistema 1 está siempre encendido — lee las caras, completa las frases, reconoce patrones. El Sistema 2 es reacio y costoso — se engancha cuando el Sistema 1 señala un mismatch o cuando lo forzamos.

El aprendizaje es una actividad del Sistema 2. Pero el Sistema 1 siempre busca atajos, y un LLM proporciona el atajo perfecto: una respuesta tan fluida que el Sistema 2 nunca se engancha. El learner se siente enseñado; nada ha sido enseñado.

La pedagogía de Fluera está construida para forzar el enganche del Sistema 2. La escritura a mano es lenta a propósito. La evaluación de confianza interrumpe la lectura de superficie. Los prompts socráticos se niegan a ser System-1-ready. El roce es el hábitat del Sistema 2.
