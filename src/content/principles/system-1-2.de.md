Kahnemans *Schnelles Denken, langsames Denken* (2011) hat Jahrzehnte seiner Arbeit mit Amos Tversky synthetisiert. System 1 ist immer eingeschaltet — es liest Gesichter, vervollständigt Sätze, erkennt Muster. System 2 ist widerwillig und kostspielig — es engagiert sich, wenn System 1 eine Diskrepanz signalisiert oder wenn wir es erzwingen.

Lernen ist eine Tätigkeit von System 2. Aber System 1 sucht stets Abkürzungen, und ein LLM liefert die perfekte Abkürzung: eine so flüssige Antwort, dass System 2 sich nie engagiert. Die lernende Person fühlt sich unterrichtet; nichts wurde gelernt.

Die Pädagogik von Fluera ist gebaut, um das Engagement von System 2 zu erzwingen. Handschrift ist absichtlich langsam. Die Konfidenz-Bewertung unterbricht die Oberflächenlektüre. Sokratische Prompts weigern sich, System-1-tauglich zu sein. Die Reibung ist der Lebensraum von System 2.
