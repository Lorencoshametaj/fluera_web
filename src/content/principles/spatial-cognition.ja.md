John O'Keefe は 1971 年に海馬で place cells を発見しました — 動物が特定の位置を占めるときだけ発火するニューロンです。May-Britt と Edvard Moser は 2005 年に grid cells を発見しました — 嗅内皮質にあり、環境を六角格子でテッセレートするニューロンです。両者が脳の生物学的 GPS を形成します。2014 年のノーベル医学・生理学賞はこの発見を認めました。

しかし海馬はナビゲーションに限られません。同じ回路がエピソード記憶を索引付けします。*記憶すること* と *ナビゲートすること* は異なって感じられますが、基底の基盤は同一です。古代の記憶術(Memory Palace)が機能するのはこのためです — その課題に専念している回路を活用するのです。

その手法を使う記憶アスリートは一貫して対照群を上回り、効果量は d = 0.88 程度です。Fluera の無限キャンバスは、あなたが手で構築する Memory Palace であり、各ノードは同じ回路によって空間的に索引付けされます。
