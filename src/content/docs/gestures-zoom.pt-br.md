O canvas infinito torna uma coisa obviamente verdadeira e outra não tão obviamente verdadeira. Obviamente: você nunca acaba o espaço. Não tão obviamente: você pode se perder no espaço. Esta página é sobre se manter orientado.

## Gestos com um e dois dedos

| Gesto                         | O que faz                                |
|-------------------------------|------------------------------------------|
| Um dedo — arrastar            | Pan do canvas                            |
| Dois dedos — pinçar           | Zoom in e out                            |
| Dois dedos — rotacionar       | Rotaciona o canvas                       |
| Dois dedos — duplo toque      | Reseta zoom e rotação para 100%          |
| Três dedos — swipe para cima  | Modo overview (vista continente)         |
| Três dedos — toque            | Quick-select da última ferramenta usada  |

No desktop, os mesmos gestos funcionam com trackpad. Usuários de mouse: <kbd>Espaço</kbd>+arrastar para pan, <kbd>Ctrl</kbd>+scroll para zoom.

## A caneta e o dedo são diferentes

Por padrão, a caneta desenha e o dedo navega. Você pode inverter isso em **Configurações → Canvas → Mapeamento de input** se preferir, mas recomendamos ficar com o padrão — mantém a sua mão apoiada na tela sem se preocupar com marcas perdidas.

## Modo overview

Swipe para cima com três dedos (ou <kbd>Cmd</kbd>+<kbd>0</kbd> no desktop) afasta o zoom para mostrar o canvas inteiro num relance. Esta é a **vista continente**. É a vista que faz o palácio da memória funcionar — deixa você ver bairros de conceitos se formarem ao longo do tempo.

Toque em qualquer região no modo overview para voltar o zoom àquela posição em um nível razoável.

## Seções nomeadas e bookmarks

Se o seu canvas estiver ficando grande, use **Seção** para marcar regiões nomeadas. Uma seção é apenas uma área retangular nomeada — toque no rótulo dela na lista de canvas para pular para lá.

Bookmarks (<kbd>Cmd</kbd>+<kbd>B</kbd>) salvam um zoom+posição específicos e te deixam voltar depois.

## O que não fazer

- **Não gaste dez minutos arrumando antes de começar a escrever.** O layout espacial emerge do conteúdo, não o contrário.
- **Não delete clusters antigos para "fazer faxina".** Eles são um registro do seu pensamento ao longo do tempo. Um bairro esparso é um dado.
- **Não dê zoom tão de perto a ponto de perder a pista espacial.** Se você só consegue ver um nó, está operando como um app de lista. Afaste o zoom de vez em quando para lembrar onde está.
