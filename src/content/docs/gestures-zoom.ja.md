無限キャンバスは、ある明らかな真実と、それほど明らかではない真実の両方を提示します。明らかな点: スペースを使い切ることはありません。それほど明らかではない点: スペースで迷子になることがあります。このページは、方向感覚を保つことについてです。

## 1本指と2本指のジェスチャー

| ジェスチャー                  | 動作                                     |
|-------------------------------|------------------------------------------|
| 1本指 — ドラッグ              | キャンバスをパン                         |
| 2本指 — ピンチ                | ズームイン・アウト                       |
| 2本指 — 回転                  | キャンバスを回転                         |
| 2本指 — ダブルタップ          | ズームと回転を100%にリセット             |
| 3本指 — 上にスワイプ          | 概観モード (大陸ビュー)                   |
| 3本指 — タップ                | 直前に使用したツールのクイック選択       |

デスクトップでは、トラックパッドで同じジェスチャーが機能します。マウスユーザー: <kbd>Space</kbd>+ドラッグでパン、<kbd>Ctrl</kbd>+スクロールでズーム。

## ペンと指は別物

デフォルトでは、ペンが描画し、指がナビゲーションします。お好みであれば**設定 → キャンバス → 入力マッピング**で逆にできますが、デフォルトのままを推奨します — 失われたマークを心配せずに手を画面に乗せておくことができます。

## 概観モード

3本指で上にスワイプ (またはデスクトップでは<kbd>Cmd</kbd>+<kbd>0</kbd>)すると、ズームアウトしてキャンバス全体を一目で表示します。これが**大陸ビュー**です。メモリパレスを機能させるビューであり、概念の街区が時間とともに形成されるのを見ることができます。

概観モードで任意の領域をタップすると、適切なレベルでその位置にズームバックします。

## 名前付きセクションとブックマーク

キャンバスが大きくなってきたら、**セクション**を使って名前付きの領域をマークします。セクションは単に名前付きの矩形領域です。キャンバスリスト内のラベルをタップしてジャンプします。

ブックマーク (<kbd>Cmd</kbd>+<kbd>B</kbd>)は、特定のズーム+位置を保存し、後で戻ることができます。

## やってはいけないこと

- **書き始める前に10分間整頓するのに費やさないでください。** 空間レイアウトはコンテンツから生まれるものであり、その逆ではありません。
- **「片付け」のために古いクラスタを削除しないでください。** それらは時間をかけたあなたの思考の記録です。まばらな街区はそれ自体がデータです。
- **空間的な手がかりを失うほどズームインしないでください。** 1つのノードしか見えないなら、あなたはリストアプリのように操作しています。時々ズームアウトして自分がどこにいるかを思い出してください。
