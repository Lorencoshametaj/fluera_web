Fluera には13種類のブラシエンジンが付属しています。それぞれが完全なレンダリングパイプライン (筆圧、傾き、速度、ブラシごとのフラグメントシェーダー)であり、テクスチャの差し替えではありません。多く感じるかもしれませんが、実際には学習のほとんどは1つか2つのブラシで行われます。

## 短い答え

- ノートを取る場合: **万年筆**または**ボールペン**。両方とも速く、読みやすく、寛容です。
- PDFに注釈を付けたり、概念をハイライトする場合: **蛍光ペン**。
- 幾何学的でクリーンな線で図を描く場合: **テクニカルペン**。
- 素早いビジュアルメタファーやイラスト: **鉛筆** (軽いストローク)または**チャコール** (ドラマチックなストローク)。

それ以外はすべて特定の状況向けです。考えすぎないでください。

## 13種類

| ブラシ              | 適した用途                                          | メモ                                          |
|---------------------|---------------------------------------------------|------------------------------------------------|
| 万年筆              | 長文の手書き                                       | 傾きに反応、自然なインクフロー                  |
| ボールペン          | 素早いノート、均一な線                             | 筆圧感知の太さ                                  |
| 鉛筆                | 柔らかく寛容なストローク                           | 粒状のテクスチャ、筆圧によるグラデーション      |
| テクニカルペン      | CADスタイルの図、精密な幾何学                      | あえて筆圧感知ゼロ                              |
| マーカー            | 角度のあるタイトル、ハイライト                     | フラットなストローク、鮮やかな色                |
| 蛍光ペン            | オーバーレイハイライト (不透明度20–30%)            | オーバーレイモードでブレンド                    |
| チャコール          | 表現豊か、粗いテクスチャ                           | ぼかし効果、エッジのにじみ                      |
| 油彩                | 絵画的なストローク、層になった色                   | ウェットエッジブレンド、エッジの暗化            |
| 水彩                | 柔らかい層、拡散                                   | 顔料シミュレーション                            |
| 淡彩                | ウェットオンペーパーの感触                         | エッジでの色のにじみ                            |
| スプレー            | パーティクルエフェクト、シャドウ                   | エッジの歪み                                    |
| ネオン              | 発光する注釈                                       | 外部グロー、蛍光彩度                            |
| 消しゴム            | ピクセル単位の削除、削除ツールではありません       | 履歴を意識 — 下のマークを表示                   |

## なぜエンジンが重要か (簡潔に)

Fluera のエンジンは、iOS と macOS で15ms未満のレイテンシで筆圧、傾き、速度のシグナルを保持します。これは単なるプロダクトのこだわりではありません。<a href="/ja/science/authors/van-der-meer">van der Meer の EEG 研究</a>は、手書きがタイピングでは関与しない感覚運動ネットワークを動員することを示しており、これらシグナルの豊かさはキャプチャの忠実度に応じてスケールします。

実用的な観点から: ストロークを失ったり、筆圧の下で遅延したり、傾きデータを平坦化したりするキャンバスは、手書きノートを認知的に優れたものにしているまさにそのシグナルを捨てています。市販の選択肢で基準を満たすものがなかったため、私たちはエンジンをゼロから構築しました。

## ブラシをカスタマイズする

ツールバーの任意のブラシを長押ししてください。次のオプションを含む小さなパネルが開きます。

- **サイズ** (0.5pt – 12pt)
- **不透明度** (10% – 100%)
- **スタビライゼーション** (なし / 柔らかい / しっかり) — 個性を殺すことなくジッターを滑らかにします
- **筆圧カーブ** — 軽い筆圧と強い筆圧への反応を調整します

変更はキャンバスごと (グローバルではなく)なので、あるノートでは柔らかい鉛筆、別のノートではシャープな鉛筆を持つことができます。
