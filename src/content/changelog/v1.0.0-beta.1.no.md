Vi åpner portene, sakte, for første kohort av Fluera-betaen. iOS (iPadOS) og macOS kommer først — Android, Windows og Linux følger etter hvert som vi jobber oss gjennom alfa-backloggen.

**Hva som er inkludert**

- Uendelig canvas med 13 trykksensitive pensler
- Komplett 12-stegs læringssyklus, med alle de fire strukturelle funksjonene aktive
- Planlegger for distribuert repetisjon med tilpassede FSRS-vekter
- Local-first-lagring kryptert med SQLCipher, opt-in sky-synkronisering
- P2P-samarbeid kryptert ende-til-ende (Visit, Teaching, Duel-moduser)

**Hva som ikke er inne ennå**

- Time Travel audio-sync fortsatt i alfa, kun på iOS
- Android-release i neste beta-syklus
- Dashboard for revisjonsspor for institusjoner på roadmapen

**Hvordan komme inn**

Meld deg på [beta-ventelisten](/beta). Vi onboarder i bølger på rundt femti i uken og leser hvert svar.
