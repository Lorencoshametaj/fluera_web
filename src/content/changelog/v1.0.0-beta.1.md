We are opening the gates, slowly, to the first Fluera beta cohort. iOS (iPadOS) and macOS ship first — Android, Windows and Linux follow as we work through the alpha backlog.

**What's included**

- Infinite canvas with 13 pressure-sensitive brushes
- Full 12-step learning cycle, with all four pillar features enabled
- Spaced repetition scheduler with personalised FSRS weights
- Local-first SQLCipher-encrypted storage, opt-in cloud sync
- End-to-end encrypted P2P collaboration (Visit, Teaching, Duel modes)

**What's not yet**

- Time Travel audio-sync is still alpha on iOS only
- Android release in the next beta cycle
- Education audit-log dashboard (Phase 2)

**How to get in**

Join the [beta waitlist](/beta). We onboard in batches of around fifty a week and read every reply.
