Vi öppnar portarna, långsamt, för Flueras första betakohort. iOS (iPadOS) och macOS kommer först — Android, Windows och Linux följer när vi arbetar oss igenom alpha-backloggen.

**Vad som ingår**

- Oändlig canvas med 13 tryckkänsliga penslar
- Komplett 12-stegs lärandecykel, med de fyra strukturella funktionerna aktiva
- Schemaläggare för distribuerad repetition med personliga FSRS-vikter
- Local-first storage krypterad med SQLCipher, opt-in cloud sync
- P2P-samarbete krypterat end-to-end (Visit-, Teaching-, Duel-lägen)

**Vad som inte ännu**

- Time Travel audio-sync är fortfarande i alpha, endast iOS
- Android-release i nästa beta-cykel
- Audit trail-dashboard för institutioner finns på roadmappen

**Hur du kommer in**

Anmäl dig till [betans väntelista](/beta). Vi onboardar i vågor om cirka femtio i veckan och läser varje svar.
