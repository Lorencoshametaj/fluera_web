Ghost Map arrive dans sa première forme utilisable. Après l'étape 3 du cycle, les étudiants peuvent activer un overlay de comparaison : leur canvas contre une carte idéale générée par l'IA. Les écarts pulsent visuellement.

**Nouveautés**

- Overlay Ghost Map avec indices rouges (manquants), jaunes (mal connectés), bleus (ajoutés)
- Slider de confiance à chaque révélation socratique — active l'hypercorrection
- Flux de tentative on-canvas : les étudiants écrivent la correction à la main avant que l'idéal apparaisse
- FAB avec boutons Comparer / Révéler pendant les tentatives

**Améliorations**

- Le placement de Gemini est non déterministe ; on a ajouté un post-pass anti-overlap (espacement minimum de 350 px)
- La pipeline de dessin ne bloque plus pendant la tentative inline
- Le texte des étiquettes est maintenant ancré à une taille d'écran fixe (160 px, 1 ligne) ; les rectangles s'adaptent au canvas

**Problèmes connus**

- Gemini entasse parfois les nœuds dans un seul cluster — le post-pass les éloigne parfois de groupes liés
- Pas d'auto-détection quand un étudiant finit d'écrire une tentative (seulement confirmation manuelle)
- La finition visuelle des nœuds ghost (animations, transitions) sera améliorée en v0.10
