Vi åbner portene, langsomt, til den første kohorte i Fluera-beta'en. iOS (iPadOS) og macOS kommer først — Android, Windows og Linux følger, efterhånden som vi arbejder os igennem alpha-backloggen.

**Hvad der er med**

- Uendeligt canvas med 13 trykfølsomme pensler
- Komplet 12-trins lærings-cyklus, med de fire strukturelle funktioner aktive
- Planlægger til distribueret gentagelse med personlige FSRS-vægte
- Local-first storage krypteret med SQLCipher, opt-in cloud sync
- P2P-samarbejde end-to-end-krypteret (Visit-, Teaching-, Duel-tilstande)

**Hvad der ikke er endnu**

- Time Travel audio-sync stadig i alpha, kun på iOS
- Android-release i den næste beta-cyklus
- Audit-trail-dashboard for institutioner på roadmappet

**Sådan kommer du med**

Skriv dig op på [beta-waitlisten](/beta). Vi onboarder i bølger på omkring halvtreds om ugen og læser hvert eneste svar.
