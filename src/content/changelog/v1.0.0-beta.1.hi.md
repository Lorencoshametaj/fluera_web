हम धीरे-धीरे, Fluera की beta की पहली cohort के लिए गेट खोलते हैं। iOS (iPadOS) और macOS पहले आते हैं — Android, Windows और Linux बाद में आते हैं जैसे हम alpha backlog से होकर काम करते हैं।

**क्या शामिल है**

- 13 दबाव-संवेदनशील brushes के साथ अनंत कैनवास
- पूर्ण 12-step learning cycle, चारों संरचनात्मक features सक्रिय के साथ
- Personalized FSRS weights के साथ अंतरालित दोहराव scheduler
- SQLCipher के साथ encrypted local-first storage, opt-in cloud sync
- End-to-end encrypted P2P collaboration (Visit, Teaching, Duel modes)

**क्या अभी नहीं**

- Time Travel audio-sync अभी भी alpha में, केवल iOS पर
- Android release अगले beta cycle में
- संस्थानों के लिए audit trail dashboard roadmap पर

**कैसे शामिल हों**

[Beta waitlist](/beta) के लिए sign up करें। हम लगभग पचास प्रति सप्ताह की लहरों में onboard करते हैं और हर response पढ़ते हैं।
